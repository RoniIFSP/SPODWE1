Resoluções dos Exercícios do CSS Diner

Lista de seletores para cada um dos 26 desafios do site CSS Diner.

  1. Exercício 1: `plate` → plate
  2. Exercício 2: `bento` → bento
  3. Exercício 3: `#fancy` → #fancy
  4. Exercício 4: `plate apple` → plate apple
  5. Exercício 5: `#fancy pickle` → #fancy pickle
  6. Exercício 6: `apple.small` → apple.small
  7. Exercício 7: `bento orange.small` → bento orange.small
  8. Exercício 8: `plate, bento` → plate, bento
  9. Exercício 9: `*` → *
  10. Exercício 10: `plate *` → plate *
  11. Exercício 11: `plate + apple` → plate + apple
  12. Exercício 12: `bento ~ pickle` → bento ~ pickle
  13. Exercício 13: `plate > apple` → plate > apple
  14. Exercício 14: `orange:first-child` → orange:first-child
  15. Exercício 15: `plate:nth-child(3)` → plate:nth-child(3)
  16. Exercício 16: `bento:nth-last-child(3)` → bento:nth-last-child(3)
  17. Exercício 17: `apple:first-of-type` → apple:first-of-type
  18. Exercício 18: `plate:nth-of-type(even)` → plate:nth-of-type(even)
  19. Exercício 19: `plate:nth-of-type(2n+3)` → plate:nth-of-type(2n+3)
  20. Exercício 20: `apple:only-of-type` → apple:only-of-type
  21. Exercício 21: `bento:empty` → bento:empty
  22. Exercício 22: `apple:not(.small)` → apple:not(.small)
  23. Exercício 23: `*[for]` → *[for]
  24. Exercício 24: `bento[for="Vitaly"]` → bento[for="Vitaly"]
  25. Exercício 25: `plate[for^="Sa"]` → plate[for^="Sa"]
  26. Exercício 26: `bento[for$="ato"]` → bento[for$="ato"]
